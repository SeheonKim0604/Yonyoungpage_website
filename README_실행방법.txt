========================================
연영회 홈페이지 실행 방법
========================================

방법 1: 자동 실행 (가장 간단)
----------------------------------------
1. "start.bat" 파일을 더블클릭하세요.
   - 처음 실행 시 자동으로 패키지를 설치합니다.
   - 서버가 시작되면 브라우저에서 http://localhost:3000 으로 접속하세요.

방법 2: 수동 설치 후 실행
----------------------------------------
1. "install.bat" 파일을 더블클릭하여 패키지를 먼저 설치하세요.
2. 설치가 완료되면 "start.bat" 파일을 더블클릭하세요.

방법 3: 명령 프롬프트(cmd) 사용
----------------------------------------
1. Windows 키 + R을 누르고 "cmd" 입력 후 Enter
2. 다음 명령어들을 순서대로 입력:

   cd "C:\Users\kimse\Desktop\김세헌\연세대학교 학교생활 관리\3학년 겨울\연영회 홈페이지\yeonyeonghoe-website"
   npm install
   npm run dev

3. 브라우저에서 http://localhost:3000 으로 접속하세요.

========================================
문제 해결
========================================

Q: "Node.js가 설치되어 있지 않습니다" 오류가 나옵니다.
A: Node.js를 설치하고 컴퓨터를 재시작하세요.
   다운로드: https://nodejs.org/

Q: "포트 3000이 이미 사용 중입니다" 오류가 나옵니다.
A: 다른 프로그램이 3000번 포트를 사용하고 있습니다.
   해당 프로그램을 종료하거나, package.json의 dev 스크립트를 수정하세요.

Q: 배치 파일이 실행되지 않습니다.
A: 파일을 우클릭하고 "관리자 권한으로 실행"을 선택하세요.

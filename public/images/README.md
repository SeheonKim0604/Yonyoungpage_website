# 이미지 폴더 구조

이 폴더에는 웹사이트에서 사용하는 모든 이미지 파일이 저장됩니다.

## 폴더 구조

```
images/
├── banner/          # 배너 이미지 (홈페이지 히어로 섹션)
├── gallery/         # 갤러리 이미지 (작품 갤러리)
├── works/           # 작품 이미지 (작품 상세 페이지)
└── exhibitions/     # 전시회 이미지 (전시회 페이지)
```

## 데이터 업데이트 방법

1. 해당 폴더에 이미지 파일을 추가합니다.
2. 웹사이트의 관리자 모드에서 이미지를 업로드하거나, `src/config/` 내의 해당 JSON 파일을 열어 이미지 경로를 업데이트합니다.

### 예시

배너 이미지를 추가하려면:
1. `banner/` 폴더에 `banner1.jpg` 파일을 추가
2. `src/config/banner.json` 파일에 `"/images/banner/banner1.jpg"` 추가

활동 기록을 추가하려면:
1. `gallery/` 폴더에 이미지를 추가
2. 웹사이트의 활동 기록 페이지에서 '+' 버튼을 눌러 업로드하거나, `src/config/activities.json` 파일에 정보를 추가합니다.

## 이미지 권장 사양

- **배너 이미지**: 1920x1080px 이상, 가로형
- **갤러리 이미지**: 최소 1200px 너비, 비율 4:3 또는 3:2 권장
- **파일 형식**: JPG, PNG, WebP
- **파일 크기**: 각 이미지당 2MB 이하 권장

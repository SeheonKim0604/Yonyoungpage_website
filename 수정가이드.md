# 홈페이지 수정 가이드

이 문서는 홈페이지의 각 부분을 수정하는 방법을 안내합니다.

## 📁 파일 구조

```
src/
├── app/                    # 페이지 파일들
│   ├── page.tsx           # 홈페이지 (메인)
│   ├── about/
│   │   └── page.tsx       # ABOUT 페이지
│   ├── archive/
│   │   └── page.tsx       # ARCHIVE 페이지 (구 WORKS)
│   ├── exhibitions/
│   │   └── page.tsx       # EXHIBITIONS 페이지
│   ├── recruiting/
│   │   └── page.tsx       # RECRUITING 페이지
│   └── donate/
│       └── page.tsx       # DONATE US 페이지
├── components/            # 재사용 컴포넌트
│   ├── Navigation.tsx    # 상단 네비게이션 바
│   ├── Hero.tsx          # 히어로 섹션 (배너)
│   └── Gallery.tsx       # 갤러리 컴포넌트
    └── config/
        ├── banner.json       # 배너 이미지 설정
        ├── activities.json   # 활동 기록 데이터
        ├── exhibitions.json  # 전시회 데이터
        ├── photographers.json # 사진가 데이터
        ├── linktree.json     # 링크트리 데이터
        └── about.json        # 소개 및 연혁 데이터
```

---

## 🏠 홈페이지 (메인 페이지)

### 파일 위치
`src/app/page.tsx`

### 수정 가능한 내용

1. **섹션 제목 변경**
   ```tsx
   <h2>최신 작품</h2>  // 이 부분을 원하는 텍스트로 변경
   ```

2. **"모든 작품 보기" 버튼 텍스트 변경**
   ```tsx
   <Link href="/archive" className={styles.viewAllLink}>
     모든 작품 보기 →  // 이 부분을 원하는 텍스트로 변경
   </Link>
   ```

3. **미리보기 작품 개수 변경**
   ```tsx
   const previewImages = (galleryImages.gallery || []).slice(0, 6)
   // slice(0, 6)의 6을 원하는 개수로 변경
   ```

## 🎨 히어로 섹션 (상단 배너)

### 파일 위치
`src/components/Hero.tsx`

### 수정 가능한 내용

1. **제목 변경**
   ```tsx
   <h1>연영회</h1>  // 이 부분을 원하는 제목으로 변경
   ```

2. **부제목 변경**
   ```tsx
   <p>사진으로 세상을 기록합니다</p>  // 이 부분을 원하는 부제목으로 변경
   ```

3. **배너 이미지 변경**
   - 웹사이트 내에서 직접 업로드하거나,
   - `src/config/banner.json` 파일을 열어서 이미지 경로를 추가/수정합니다.
   ```json
   [
     "/images/banner/banner1.jpg",
     "/images/banner/banner2.jpg"
   ]
   ```

4. **슬라이더 속도 변경** (5초마다 변경)
   ```tsx
   }, 5000)  // 5000을 원하는 밀리초로 변경 (예: 3000 = 3초)
   ```

## 📋 네비게이션 바

### 파일 위치
`src/components/Navigation.tsx`

### 수정 가능한 내용

1. **로고 텍스트 변경**
   ```tsx
   <Link href="/" className="logo">
     연영회  // 이 부분을 원하는 텍스트로 변경
   </Link>
   ```

2. **메뉴 항목 추가/제거/변경**
   ```tsx
   const navItems = [
     { href: '/about', label: 'ABOUT' },
     { href: '/archive', label: 'ARCHIVE' },
     // 여기에 새로운 메뉴 항목 추가
     { href: '/new-page', label: 'NEW PAGE' },
   ]
   ```

## 🖼️ 이미지 및 데이터 관리

### 데이터 설정 파일들
**위치**: `src/config/` 폴더 내 JSON 파일들

각 페이지의 데이터는 별도의 파일로 분리되어 관리됩니다.

### 배너 이미지 (`banner.json`)
```json
[
  "/images/banner/banner1.jpg",
  "/images/banner/banner2.jpg"
]
```

### 갤러리/활동 기록 (`activities.json`)
```json
[
  {
    "id": 1,
    "title": "활동 제목",
    "date": "2024.01.01",
    "coverImage": "/images/gallery/photo1.jpg",
    "images": ["/images/gallery/photo1.jpg"]
  }
]
```

### 전시회 정보 추가/변경
```json
{
  "exhibitions": [
    {
      "id": 1,
      "title": "2024 연영회 전시",
      "date": "2024.12.01 - 2024.12.31",
      "location": "갤러리 이름",
      "image": "/images/exhibitions/exhibition1.jpg",
      "description": "전시 설명입니다."
    }
  ]
}
```

## 💡 수정 후 확인 방법

1. 웹사이트의 관리자 모드에서 수정 시 즉각 반영됩니다.
2. JSON 파일을 직접 수정한 경우, 브라우저를 새로고침(F5)하면 변경사항이 즉시 반영됩니다. (캐시가 남은 경우 강력 새로고침 Ctrl+F5를 권장합니다.)

---

## ⚠️ 주의사항

1. **파일 저장**: 수정 후 반드시 파일을 저장하세요 (Ctrl+S)
2. **문법 오류**: 괄호, 따옴표 등을 빠뜨리지 않도록 주의하세요
3. **이미지 경로**: 이미지 파일은 `public/images/` 폴더에 넣고, 경로는 `/images/...` 형식으로 작성하세요
4. **반영 확인**: 데이터를 수정한 후에는 브라우저 새로고침만으로 확인이 가능합니다. (서버 재시작 불필요)

## 📝 예시: ABOUT 페이지 텍스트 수정하기

1. `src/app/about/page.tsx` 파일을 엽니다
2. 원하는 텍스트를 찾아서 수정합니다:
   ```tsx
   <p>
     연영회는 사진을 통해 세상을 기록하고 표현하는 동아리입니다.
     // 이 부분을 원하는 내용으로 변경
   </p>
   ```
3. 파일을 저장합니다 (Ctrl+S)
4. 브라우저에서 자동으로 업데이트된 내용을 확인합니다
